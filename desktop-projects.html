Sharp Converter

Current Stable Release: v0.4
Platforms: Windows 10

Sharp Converter (a.k.a #Converter) is a simple console tool built using C#. The tool allows you to convert between different number systems, IEEE 754 single-precision standard, and Unicode. The purpose of this project was to implement my own conversion algorithims without using the help of any predefined methods available in the .NET library. The tool includes its own command parsing interface.
